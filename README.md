# Il Frutto Proibito - Erboristeria Mondovì

Sito e-commerce per l'erboristeria "Il Frutto Proibito" di Mondovì (CN).

## Installazione su Hostinger

1. Importa questo repository su Hostinger (App web Node.js)
2. Aggiungi le variabili d'ambiente (vedi .env.example)
3. Importa `database-mysql.sql` in phpMyAdmin
4. Avvia l'app

## Variabili d'ambiente richieste

- **NODE_ENV** = production
- **PORT** = 3000
- **DATABASE_URL** = mysql://utente:password@host:3306/database
- **SESSION_SECRET** = (stringa casuale lunga)
